Infant Birthweight Analysis

Data analysis and visualization using R, writeup using LaTeX software.

In this research project, I analyzed public data to look for effects on infant birth weight. The goal of the project was to see how infant birth weight is impacted, looking at several different variables. Although some hypotheses were revealed to be insignificant, there were surprising results, like the fact that father's age plays a significant part. The model was verified to be unbiased through multiple linear regression analysis.

NBA Analysis

Data analysis and visualization using Python.

In this project, I looked at NBA API data in order to determine how the game of basketball has changed over time. The goal of this project was an open-ended exploration of NBA data. Using multiple data sources, I was able to illustrate changes in the league's history, I explored how the game has changed, as well as which players had the most impact.

College Health Study

Data analysis using SQL, interviews, and surveys. Data visualization using Tableau.

This study analyzed changes in student health and nutrition data when coming to college. The goal of this study was to observe potential changes in college students health habits after coming to college. Many students shared sentiments regarding changes in their health, including a lack of time required to exercise more. Full report here.
